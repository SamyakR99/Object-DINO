# Object-DINO 🦕

Welcome to the **Object-DINO** repository. This project focuses on leveraging DINO-based features for advanced object analysis, discovery, and mitigation of hallucinations in Multi-modal Large Language Models (MLLMs).

## 🚀 Overview

Object-DINO is designed to provide a robust framework for:
- **Unsupervised Object Discovery**: Identifying objects in scenes without explicit labels.
- **MLLM Hallucination Mitigation**: Using grounded object analysis to reduce hallucinations in vision-language models.
- **Detailed Analysis**: In-depth evaluation of feature distributions and boundary purity.
